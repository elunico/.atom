JPEdit

A simple text editor written in Java 10 using JavaFX. This was an excercise in JavaFX and GUI design. It is a fairly simple text editor that has autosave, recovery features, multiple windows, open/save/duplicate, copy, cut, pase, find, replace, and insert time/date features. It was modeled after notepad.exe on Windows 10. JPEdit also supports a very rudamentary plugin system. A person can implement the JPEditPlugin interface on a class and by adding the class to a jar, load functionality into JPEdit. This system is fairly simple, but does enable some substantial functionality to be side-loaded.

SocDraw

SocDraw is a collaborative drawing application for real-time shared canvas drawing between multiple clients. The application is based on the idea of a room. A room is simply a canvas, all associated drawing data, and the clients connected to the room. Rooms are created on demand when a user chooses to do so from the home page. Rooms are identified by a sequence of 4 random words. Users can share this sequence with others who can use it to join the room and draw on a shared canvas in real time. When all members leave a room there is a short grace period where reconnections are permitted before the room and all associated drawing data is destroyed. Clients who connect to an existing room will have the current state of the canvas transmitted and immediately appear when entering. This program is made using The Processing Foundation's p5.js library for working with the canvas. The collaborative aspect is built using socket.io.

The app can be viewed and used here on Heroku.

The Redistricter

The Redistricter is a single page web app whose purpose is to optimize the shape, size, and contents of US Congressional districts in a way that maximizes certain characteristics as determined by the user. It consists of a front-end using jQuery, leaflet, and bootstrap, to display a map of states and draw precinct and district data to it. It also presents the user with several options for optimization goals in forming the districts such as compactness of shape, party fairness, and equal distribution of population. The back-end is written in Java using Apache Tomcat and JDBC and MySQL for persistence. The goal of this application was to be able to systematically draw district lines to prevent gerrymandering and have districts which are drawn based on objective criteria instead of to force a partisan advantage. It was done in a team of 4 over the course of the Fall 2018 semester at Stony Brook University for CSE 308.

This project is not currently open source.

ClipboardManager

ClipboardManager is a program written in Java to manage clipboard history. It allows the user to keep track of their clipboard history and recopy old entries. It supports rich data as well as conversion to plain text where possible. It allows the user to save clipboard history as well.

Duplicate File Manager

DuplicateFileManager is a program written in Java and Kotlin that can recursively scan any number of directories and detect duplicate files that exist between them as well as within the directories themselves and subdirectories if done recursively. This is accomplished by hashing the files. A number of hashing methods are available to the user to select. Once hashed, the user can add more directories to check for duplicates, remove ones that have been scanned along with their duplicates from view, open files, show files on disk, delete files, delete all duplicates of a file, and export the results to CSV. The results are presented as a filename, path, and the hash of the file. This allows sorting by hash, thus placing all duplicate files next to each other, or sorting files by name for convenient searching.
